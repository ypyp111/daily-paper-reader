## 摘要
我们提出了两种仅利用表面形变测量值来估算浅水流动中底形的方法。其中一种基于物理信息神经网络 (PINNs)，另一种基于伴随状态法 (Adjoint State Method)。我们在的一维和二维情况下使用合成数据对这两种方法进行了测试。结果表明，这两种方法不仅能成功重建底形，还能重建表面流速。此外，两者在合理的噪声和数据稀疏程度下均表现出鲁棒性。

## Abstract
We present two methods to estimate bottom topography in a shallow water flow using only surface deformation measurements. One is based on Physics-Informed Neural Networks (PINNs) and the other on the Adjoint State Method. We test both methods using synthetic data in 1D and 2D cases. Both are able to successfully reconstruct not only the bottom topography but also the surface velocity. Both also show robustness against noise and data sparsity up to reasonable levels.

### 2. 核心方法论
论文对比了两种主流的数据同化与反演技术：

*   **物理信息神经网络 (PINNs)**：
    *   **核心思想**：将物理定律（SWE）作为正则化项引入神经网络的损失函数中。
    *   **技术细节**：使用多层感知器（MLP）参数化速度场 $u$、水深 $h$ 和底形 $h_b$。损失函数由两部分组成：数据损失（预测值与观测值的误差）和物理残差损失（动量方程和连续性方程的残差）。
    *   **优势**：无需定义离散网格，利用自动微分（Automatic Differentiation）计算导数，对数据稀疏性有较强的鲁棒性。
*   **伴随状态法 (ASM)**：
    *   **核心思想**：一种基于变分法的经典优化手段。
    *   **技术细节**：构造拉格朗日泛函，将物理方程作为约束条件。通过求解前向物理方程和反向伴随方程来计算损失函数对底形参数的梯度，进而利用优化算法（如 L-BFGS）迭代更新底形估计。
    *   **优势**：计算效率较高，能够捕捉到地形中的高频细节。
