# Physics-Guided Dimension Reduction for Simulation-Free Operator Learning of Stiff Differential–Algebraic Systems 

Huy Hoang Le , Haoguang Wang , Christian Moya , Marcos Netto , Senior Member, IEEE, 

and Guang Lin* 

Abstract —Neural surrogates for stiff differential–algebraic equations (DAEs) face two barriers: soft-constraint methods leave algebraic residuals that stiffness amplifies into errors, and hard-constraint methods require trajectory data from stiff integrators. We introduce an extended Newton implicit layer that enforces 

g = 0 and quasi-steady-state reduction ffast ≈ 0 in a single differentiable solve. Given slow-state predictions from a physics-informed DeepONet, this layer recovers all fast and algebraic states exactly, removes the per-window stiffness-amplification pathway, and reduces the output dimension to only the slow states. Implicit Function Theorem gradients capture a stiffness-scaled coupling term absent from penalty methods. Cascaded implicit layers extend this to multi-component systems with provable convergence. On a grid-forming inverter DAE (21 states, stiffness ratio ≈ 4,712 ), extended Newton (7 outputs, 1.42% er-ror) outperforms penalty (39.3%), standard Newton (57.0%), and augmented Lagrangian/feedback linearization baselines, which diverged. Two independently trained models compose into a 44-state system without retraining (0.72%–1.16% error, ∥g∥ = 0 ). Conformal prediction confirms 90% in-distribution coverage with automatic out-of-distribution detection. 

Index Terms —Differential–algebraic equations, physics-informed neural networks, operator learning, implicit layers, singular perturbation.

Neural surrogates approximate the solution operator (x0, μ) 7 → x(t) orders of magnitude faster than implicit integrators, but on stiff systems they face a central challenge: satisfying algebraic constraints without trajectory data. 

A. The Data–Constraint Dilemma 

Learning DAE surrogates forces a trade-off between data efficiency and constraint satisfaction: 

Path A: Soft constraints, no data. Physics-informed neural networks (PINNs) penalize the algebraic residual ∥g∥2 during training (e.g., DAE-PINN [4], augmented-Lagrangian PINNs (AL-PINNs) [5]). Although this avoids simulation data, it empirically leaves ∥g∥ ∼ 10 −2 to 10 −5. In stiff systems, these small residuals amplify into large derivative errors, degrading convergence for fast states. 

Path B: Hard constraints, need data. Newton/Karush– Kuhn–Tucker (KKT) layers enforce g = 0 exactly [6]–[8], but require trajectory data from stiff integrators, the very bottleneck surrogates aim to bypass. 

Path C: Hard constraints, data-free, poor scalability. 

One could embed a Newton solver within a physics-informed operator network. However, for multi-component systems the monolithic Newton system grows cubically with the number of components, making training prohibitive. To our knowledge, no existing method takes this path. 

B. The Dimension Reduction Opportunity 

Resolving the data–constraint dilemma alone leaves phys-ical structure unexploited. Neural operator surrogates predict trajectories one window at a time: the predicted state at t = Tw

The nf fast states follow from the system’s quasi-steady-state dynamics. Fast states (e.g., converter currents) settle to quasi-steady-state (QSS) within each window, satisfying ffast ≈ 0 at the boundary, providing additional equations for Newton analo-gous to g = 0. Predicting fast states is unnecessary and forces the neural network to resolve O(1 /κ )-timescale oscillations. The neural network predicts only the slow states; Newton recovers the rest to machine precision at every collocation point. 

C. Contributions 

We propose a unified framework in which a Newton solver resolves all algebraic and quasi-steady-state constraints, leav-ing the network to learn only the irreducible slow dynamics. To our knowledge, this is the first approach to simultaneously achieve hard constraint enforcement, simulation-free training, and stiffness-aware dimension reduction for DAEs. (i) Extended Newton implicit layer with singular pertur-bation reduction. We unify algebraic constraint enforce-ment ( g = 0) and quasi-steady-state reduction ( ffast ≈ 0)within a single differentiable implicit layer, embedded in a PI-DeepONet [9]. This layer solves the combined system [ffast ; g] = 0 for all fast and algebraic states, given only the slow states from the neural network. Three consequences follow: (a) the network output di-mension reduces from nx to ns; (b) the per-window κ-amplification pathway is removed (Proposition 3); and (c) the Implicit Function Theorem (IFT) gradients capture a constraint coupling term scaled by κ that is structurally absent from all penalty methods. No trajectory data is required. (ii) Cascaded implicit layers for multi-component scala-bility. For systems with hierarchical algebraic structure (triangular local Jacobians, sparse network coupling), the extended Newton system decomposes into parallel local solutions plus a small network-level Newton step, with provable linear convergence (Theorem 1). By design, each component operator is self-contained, ac-cepting coupling variables as inputs while Newton solves only local equations. A. Soft-Constraint Methods for DAEs 

Soft-constraint approaches penalize the algebraic resid-ual ∥g∥2 during training. DAE-PINN [4] combines implicit Runge–Kutta time-stepping with penalty-weighted residuals, whereas AL-PINNs [5] replaces the fixed penalty with an aug-mented Lagrangian scheme. Trust-region sequential quadratic programming (SQP) [14] achieves 100 × lower residuals than standard penalties. At the system level, PINNSim [1] trains component-level PINNs separately and couples them through an external Newton step; however, this non-differentiable cou-pling blocks gradient flow through the algebraic solution. Koch et al. [12] train a differentiable algebraic surrogate ˆz = hϕ(x)

jointly with the ordinary differential equation (ODE) network via operator splitting. This avoids penalty tuning but still leaves ∥g(x, ˆz)∥ > 0. On stiff systems, even small residuals are amplified through the fast dynamics. Feedback linearization (FL) [13] recasts the physics-informed loss with a control-inspired multiplier update, achieving lower boundary errors but still driving ∥g∥ → 0

only asymptotically. All soft-constraint methods leave ∥g∥ > 0, and on stiff systems the residual is amplified into the fast-state dynamics by a factor proportional to κ.

B. Hard-Constraint Methods for DAEs 

Hard-constraint methods embed a solver in the forward pass to enforce g = 0, with gradients via the IFT, an approach proposed by Donti et al. [6] for static constrained optimization. For dynamical systems, methods include manifold projec-tion [7], KKT projection [8], predict–project [10], stabilized neural differential equations (Baumgarte stabilization) [15], and simultaneous collocation via the Ipopt nonlinear pro-gramming solver [16]. All share two limitations: they require trajectory data and treat all constraints monolithically, without exploiting timescale separation. 

C. Operator Learning for Dynamical Systems 

PI-DeepONet [9] extends DeepONet [17] with physics-informed training for ODEs and PDEs. Recent extensions include PI neural operators for power system dynamics [18] and DeepONet-Grid-UQ [11] with Bayesian uncertainty quan-tification. All address ODE systems; none combines operator learning with hard DAE constraints. Our extended Newton layer bridges this gap by exploiting timescale separation. 

D. Singular Perturbation in Numerical Methods and Machine Learning 

Singular perturbation theory [19], [20] decomposes stiff ODEs into slow and fast subsystems via the QSS approxima-tion, ffast ≈ 0, enabling model reduction [21] and stable dis-cretization [22]. Ji et al. [23] apply QSS as a preprocessing step to reduce stiffness before PINN training. In machine learning, Lee and Temam [24] propose fast-slow neural networks that encode timescale separation architecturally from trajectory data. Caldana et al. [25] address stiffness in neural ODE surrogates through data-driven time reparametrization. All three are data-driven and do not enforce algebraic constraints. Our extended Newton layer instead embeds the QSS solution within a differentiable implicit layer, enforcing constraints exactly without manual equation rewriting. 3

E. Uncertainty Quantification for Neural Surrogates 

Conformal prediction (CP) [26], [27] provides distribution-free prediction sets from held-out residuals. Extensions include conformalized quantile regression [28], adaptive conformal inference [29], and multi-step time-series forecasting [30]. Existing uncertainty quantification for power-system surro-gates is primarily Bayesian [11]; no prior work applies CP to hard-constrained DAE surrogates. In our framework, Newton-solved states inherit uncertainty from slow-state predictions, unlike standard time-series CP [30]; we use CP as a diagnostic (§V-E). III. P ROBLEM FORMULATION

V. E XPERIMENTS 

We validate on power system DAEs of increasing com-plexity. The central experiment (Exp 2) compares extended Newton ( ns = 7 ) against four baselines on a stiff 21-state system ( κ ≈ 4,712 ). 

A. Test Systems 

System A: Single-machine infinite bus (SMIB). nx = 2 ,

nz = 1 , κ ∼ 10 (mild stiffness). Both states are slow, so only the base Newton layer is needed.

Role of IFT gradients. The Lκ-scaled coupling in (11) is absent from penalty methods and is the dominant gradient component on stiff systems. Extended Newton further removes the need to predict fast states, yielding 28 ×–40 × improve-ment. 

QSS validity and robustness. The method requires Tw ≫10 

τfast and stable fast eigenvalues. For the GFM, Tw/τ fast ≈ 10 ;accuracy is robust across Tw = 1–10 ms but degrades at 20 ms (Table VII). Zhao et al. [22] independently validated this timescale separation. 

Limitations. (a) The partition must be known a priori (eigenvalue clustering can automate this; misspecification costs 6.9×, Table VII). (b) QSS error is nonzero in the first window after a disturbance. (c) Cascaded convergence is validated for N = 2 ; larger systems need sparse network Newton. (d) CP provides marginal, not conditional, coverage. 

Practical considerations. Analytical Jacobians avoid au-tomatic differentiation (AD) overhead; for black-box mod-els, AD substitutes at higher cost. Efficient use of graphics processing units (GPUs) requires batched solves and mixed-precision arithmetic; our implementation handles nf +nz ≤ 20 

with negligible overhead. Training cost is 4.7 h on a single A100 GPU (200k epochs); inference completes a 0.5 s rollout in ∼0.1 s vs. ∼2–5 s for Radau IIA. The Newton-in-the-loop interface supports compositional deployment [31], with training cost scaling by the number of component classes, not system size. VII. C ONCLUSION 

This paper presented an extended Newton implicit layer that unifies algebraic constraint enforcement and QSS reduction in a single differentiable solve, enabling simulation-free operator learning for stiff DAEs. The key insight is that the network should only learn what physics cannot determine exactly: the slow dynamics. On a grid-forming inverter ( κ ≈ 4,712 ), this reduces error by 28 ×–40 × over existing methods while guaranteeing ∥g∥ = 0 . Cascaded implicit layers enable zero-shot composition of independently trained component models. Several directions remain open. First, the slow/fast partition is currently specified manually; data-driven partitioning via eigenvalue clustering or sensitivity analysis could automate this step. Second, the cascaded scheme assumes weak inter-component coupling ( ρ < 1); extending to strongly coupled networks requires full Newton with sparse linear algebra. Third, the conformal prediction diagnostic provides marginal coverage; adaptive or conditional variants [29] could tighten the bands. Finally, applying the framework to larger-scale systems (e.g., bulk power grids with hundreds of inverters) will test the scalability of both the cascaded Newton coupler and the compositional training paradigm.
